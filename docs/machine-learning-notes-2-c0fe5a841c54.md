# *第 3 章:决策树学习*

# *3.1 导言*

> *决策树学习是一种逼近离散值函数的方法，对噪声数据具有鲁棒性，能够学习析取表达式，是归纳推理的一种实用方法。*

*决策树通过从根到某个叶节点对实例进行排序来对实例进行分类，这提供了实例的分类。树中的每个节点都指定了实例的某个属性的测试，从节点向下的每个分支都对应于属性的一个可能值。*

# *3.2 决策树学习的问题集*

*   *实例由一个属性值对表示。*
*   *目标函数有一个离散的输出值。*
*   *可能需要一个分离的描述。*
*   *训练数据可能包含错误。*
*   *训练数据可能包含缺失值。*

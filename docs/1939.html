结论

在这个简短回顾的最后，我们可以通过提及一些在使用 React 钩子时需要记住的关键事情来刷新我们的记忆。

钩子不会移除类——如果我们想要或者需要，我们仍然可以使用类。

钩子规则——使用钩子时必须遵守:

☙ 只调用顶层的钩子——这基本上意味着我们不能在循环、条件和嵌套函数中调用钩子(是的，记住，钩子是函数，我们可以调用它们)。我们必须遵循钩子被调用的顺序。注意，如果使用 提供的 lint 规则 就不需要担心这个问题。

☙ 只从 React 函数中调用钩子——这意味着我们不应该从常规的 JavaScript 函数中调用钩子。相反，我们应该只从 React 函数组件或定制钩子中调用它们。

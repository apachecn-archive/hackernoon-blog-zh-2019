您将见到 [Workbench](http://workbenchdata.com/) 的创建者，这是一款无需任何编码经验就能分析大数据的工具。

> “我认为自己是一名计算记者，我告诉人们这是两件事。它使用计算技术来做报道，讲述计算和算法如何影响社会。”
> 
> “一个方向可能是将数据科学用于调查性新闻报道。另一个方向可能是，例如，研究如何使用机器学习来决定谁必须在审判前呆在监狱里。”

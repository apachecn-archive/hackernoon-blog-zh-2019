Cosmic JS 是一个现代化的无头内容管理系统，它使团队能够更快地一起构建应用程序。Cosmic JS为内容创建者提供了一个基于web的仪表板来管理内容，并为开发人员提供了API工具和资源来将他们的内容与他们的应用程序集成在一起。

在本教程中，我将演示如何使用Cosmic JS CLI安装一个由 Cosmic社区中的 Ankit Kumar 使用Angular 6、Node.js和Cosmic JS构建的 Angular认证应用。用户可以使用他们的电子邮件地址或谷歌登录注册，所有数据都存储在Cosmic JS中。我鼓励你阅读原文，看看它是如何构建的。你可以从你的Cosmic JS Bucket Dashboard安装、部署和编辑这个Angular Auth应用程序的每个部分。🔥

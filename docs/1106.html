💻黑客啤酒节挑战

这是我第一次参加由 Github 、数字海洋和 Twilio 组织和赞助的 黑客节挑战赛 。挑战的目标是在 Github 上创建至少 5 个拉请求，并赢得一件限量版 t 恤。我在 Github 上提交了 7 个 pull 请求，比如 Mozilla Components 、 Travel Mate ，还有一些我自己的，比如 MediumClap 、 EasyValidation 等等。是的，我还收到了 t 恤和漂亮的礼品。谢谢 Github、数字海洋和 Twilio。

总的来说，这是我在一条推文中对 2018 年开源活动的总结。

🎤公共演讲=社区

2018 年 6 月，我开始涉足公共演讲和当地社区贡献。我上一次做公开演讲是在 2012 年，所以获得一个机会和平台有点困难。因为本地没多少人知道我。他们可能看过我的文章或 Github 项目，但他们并不了解我个人。因此，我开始给当地的活动和社区活动组织者或技术会议等打电话，请求他们让我在他们的活动中谈谈我的经历和学习。

我在开源和 Github profile 方面的工作帮助我验证了我的经验，我得到的第一个机会是在 NED Karachi 的一个项目展览中担任评委，这是巴基斯坦工程和计算机科学领域的顶级大学之一。感谢 德夫卡斯特·巴 。

📝写作=分享知识

2018 年初，我定下了至少发表 48 篇文章(每月 4 篇)的目标，作为新年决心。我不能按时到达。我在 2018 年发表了 36 篇文章，大多是在媒体上，也有在我的网站，以及 Dev.to profile 上。这些文章帮助我每月获得约 10，000 至 15，000 次浏览，平均阅读率约为 43%。我在媒体上也有超过 1000 名粉丝，目前我有大约 1300 名粉丝。我对此非常自豪，因为在 2018 年初，我只有 200 名粉丝。所以，一年大约有 1100 个追随者。

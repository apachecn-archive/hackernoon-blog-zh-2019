2018年对我来说是伟大的一年，无论是个人还是职业。去年当我写我的第一年回顾时，我不知道这些反思和年度回顾的真正影响。但它帮助我战胜了自我欺骗综合症，让我对自己负责(至少对我的内心负责)，为自己写日志，最重要的是，看到了这些年来我作为一个人和一名开发人员是如何提高自己的。

🤔谁是瓦贾哈特·卡里姆？

从今年开始，我每年都要重新定义自己。这将有助于我在阅读之前的评论时，对自己的进步和自我完善给出看法。那么，让我们现在开始:

我叫瓦贾哈特·卡里姆。我是一个有7年多经验的android开发者，一个活跃的开源贡献者，一个写了两本书的作者，一个公众演讲者。我在Contour Software做一名全职的高级android开发人员，为 【空中飞人集团】 (加拿大)的一个企业级Android应用程序远程工作关于地图和路线查找。

📆2018年初

因此，去年在我第一年的回顾中获得了很好的反响后，我向自己承诺要记录下这一年中发生的所有事情和取得的进步，这样我就可以在下一年的回顾中发表了。但是，懒惰的我，我不能收集所有的东西。我在寻找一些应用程序，而不是像脸书、Instagram等社交媒体。让我的进步保持私密和数字化。但是我找不到好的。然后我保留了这个想法，创造了一个并保持开源。

🏠新房子=新生活方式

我的2018年的开始并不理想。这是相当累人的一次。我们最近(2017年11月)搬到了新房子，所以这是一个适应新邻居、结交新朋友、记忆新旅行计划和路线等的艰难时期。除此之外，我们还有大量的贷款和债务要偿还给所有在我们困难时期帮助我们购买梦想中的房子的人。(我们爱你们所有的❤️人，并将永远感激你们 😊️ )。

我们逐月偿还债务，牺牲了几乎所有东西，比如披萨、电影、购物等等。告诉过你这是令人精疲力尽的时间。然后一月过去了，二月一日到来了。这种让人筋疲力尽的启动，变成了“灾难启动”。

我突然被新加入的工作解雇了。

为什么？因为我的雇主疯了。还是因为当初我傻到加入那家公司。还是因为我厌倦了在之前的工作中对抗负能量和肮脏的政治。或者是因为我想更快地换工作。雇主无法完成他们投标的项目，我被提前录用了。太早了。现在我成了他们工资单上的无用资源。所以，他们友好地请我回家，并付清了我所有的欠款。那么，没有任何事先通知就礼貌地要求回家算不算礼貌？我不这么认为。

对雇主说:这不太好。它甚至不接近尼斯。你在我已经很艰难的时候破坏了我的生活。我不会忘记那可怕的一天和你带来的耻辱。至少你可以要求道歉。也许这能让我松一口气。

我在那个有毒的工作场所呆了两个月，这已经让我很沮丧了。我活得像一个行尸走肉。我已经开始找新工作了，但是在找到新工作之前，我不准备离开，以保持稳定的收入。

为了欣赏生活和美好的事物，你必须先经历不好的甚至更坏的事情。

在那家公司唯一好的事情是一个非常好的朋友，他给了我希望，积极的能量，让我的一天充满欢笑，度过了美好的时光，并给了我最美好的回忆。

对你——犯罪的伙伴( 赛义德·纳扎尔·穆罕默德 ): 虽然我们在一起工作的时间很短，而且是一个非常危险的地方。我希望我们能在更好的条件和更好的工作场所相遇，在那里我们可以互相教和学一些东西，并为Android和iOS开发出优秀的应用程序。但是让我说:非常感谢你所做的一切，包括那些长时间的午休和我们一起喝的无限量的茶。谢谢你

👻自由职业=噩梦

这是一个大胆的声明。我知道。但是，至少这些年来对我来说是真的。我认识很多优秀的人，他们从事全职或兼职的自由职业&提供服务，他们在自己灵活的时间里在家工作，收入颇丰。我已经在所有平台上尝试过太多次了，包括本地和国际客户。但是，这不是我的事。句号。我不能做自由职业者。我也不会做自由职业者。我这边已经证实了。

我已经开始找新工作了。但是在我找到新工作之前，我有一个正在进行的自由职业项目。我想完成它，并得到一些钱来保持稳定的收入。我对借钱给我的人许下了许多按时还债的承诺。但是现在游戏完全不同了。我不得不为家庭开支、水电费等安排资金。更别说偿还债务了。所以，这个自由职业项目成了我生存的手段。

但当你处于艰难时期时，通常一切都会变得颠倒。这发生在我身上。这个自由职业项目成了我的噩梦。它让我彻夜难眠。这分散了我对面试的注意力。这基本上毁了我的生活。当我拿到报酬时，已经太晚了。我告诉我的朋友和家人为什么我不能偿还债务。他们中的大多数人都非常支持和理解我，帮助我度过了这一切，不再有任何问题。但我做的更多的是从新人那里借更多的债，偿还给老人们，以延长我的时间线。

事实上变得很压抑。我一直很紧张，任何人都能从我脸上看出来。最重要的是，我有一个可怕的客户，他无限的电话，信息，不停的压力，不切实际的最后期限，以及没完没了的变化和工作。是的，你可以争辩说我没有更好地管理它或类似的事情。我告诉过你。自由职业不适合我。句号。

🔍找工作=抑郁

在那份可怕的工作之前，我已经马不停蹄地工作了4年，甚至没有更新过我的简历。所以，我基本上忘记了如何找工作，如何通过这些奇怪的面试和浪费时间的招聘过程。这些年来，我学到了很多东西，并且在Android开发方面做得更好，我的投资组合中有大约100多个应用程序。已经到了这样一个地步，当我得到任何在android上开发的需求时，很可能我已经在做类似的事情了。所以，我的Bitbucket账户变成了一个更好更个性化的StackOverflow，Github，Gists，或者任何你想寻找答案的资源。

然后我有了开源、android库和良好的Github简介的完整体验。你知道吗，我的一个开源库， EasyFlipView ，已经被全球的Android开发者下载了超过4万次。

然后，承蒙真主的恩典，我是两本全球出版的书的合著者，分别是Learning Android Intents(约300页)和Mastering Android Game Development with Untiy(约400页)，作者是 Packt Pub 。谢谢你 Packt Publishing 给我机会成为这些精彩书籍的一部分。

所以，我的印象是我已经得到了所有这些好东西，好的个人资料，Github，书籍，作品集等等。找份工作会有多难？不是我去找工作，而是有很多招聘人员在找我。我真傻，对吧？但事实证明，没人关心这些事情。他们(招聘人员)关心的只是继承的定义是什么，抽象类和接口之间有什么区别。或者我能不能在白板上写Dijkstra算法？在两个月的时间里，我接受了大约40多次采访。它变得如此令人沮丧和沮丧，以至于我不再相信自己。我的自信心空前低落。

我想:“我真的是一个开发者吗？这些年我做了什么？我做的这些库是我自己的还是我只是个假小丑？我真的有资格成为android开发者或开发者吗？”

我在另一篇文章中写了这件事，以及我是如何恢复自信并找到一份令人惊叹的工作的。

高估了“低估”

我学到的是，显然，不仅仅是我。有1000名优秀的开发人员，我该怎么说呢，是整个招聘过程失败的附带损害。招聘人员不是看实践和行动中的技能，而是看理论、GPA、所谓的纯粹定义和禁忌。最近，一个非常活跃的开源贡献者在媒体上分享了他所有糟糕招聘的可怕遭遇。

💼新的(远程)工作=快乐

所以，经过大约40+次面试，无数的笔试，在线测试，无限的问题，愚蠢的项目和任务，不眠之夜，焦虑的脸，疲惫的头脑，妻子，家人和朋友的大力支持，以及2个月的奋斗，我得到了一份很棒的工作，工资和福利很高，时间灵活，在家工作，几乎没有政治文化。

感谢 Contour Software 和Trapeze Group(加拿大)给我这个机会参与如此大规模的项目。

在这份工作之前，我的所有工作主要是关于服务提供和基于消费者的应用程序。但在Contour Software，我很幸运地为Android开发了一个企业级应用程序，并成为了real team的一部分。我们遵循严格的敏捷方法，对敏捷和scrum完全陌生，这是一次真正疯狂和具有挑战性的经历，我学到了很多新东西。

我不知道为什么我以前没有尝试过敏捷，但是现在我已经掌握了这个概念并且理解了它是如何工作的。现在，我也在我的个人项目中遵循这一点(稍后会有更多的介绍)。就像我在做一个人的scrum系统。我有一篇文章计划写关于它和我到目前为止所取得的成就。

由于在Contour Software的工作提供了灵活的时间安排和在家工作的能力，我开始更多地探索这一点。我开始享受这种自由，这让我的生活变得轻松多了，我可以更加专注和自信地关注个人、家庭和职业。此外，远程工作经常会混淆你的工作和家庭，模糊两者之间的界限，所以这也成为一个挑战。(今年，我还计划写一篇关于如何管理远程工作和个人生活的文章)。

👍开源=动机

我的整个2018年都是关于开源和为它做贡献的。我称自己为“偶然的开源贡献者”因为老实说，我是偶然进入开源的。有时候，当你害怕某样东西，你正在逃避它，那么在紧张和恐惧中，你并不真的在乎该走哪条路，你只是跑着，跑向你首先看到的任何一条路。这就是我进入开源的故事。我逃离了自由职业，最终成为了一名开源贡献者。

我开始大量创建开源项目和android库。除了作业(远程)，我只做这个。

不自由职业。

没有创业。

没有副业。

我每天花3-4个小时创建新的库，做新的实验，修复错误，回答问题，并为其他项目做贡献，如 Mozilla Mobile 等。为了更严肃地对待开源，我在Twitter 上参加了 #100DaysOfCode 挑战。每一天都工作并有所贡献是艰难的，但这是值得的，令人愉快的，也是一种人生经历。在这篇文章中，我分享了我的整个经历和我从中得到的东西。

🎤公共演讲=社区

2018年6月，我开始涉足公共演讲和当地社区贡献。我上一次做公开演讲是在2012年，所以获得一个机会和平台有点困难。因为本地没多少人知道我。他们可能看过我的文章或Github项目，但他们并不了解我个人。因此，我开始给当地的活动和社区活动组织者或技术会议等打电话，请求他们让我在他们的活动中谈谈我的经历和学习。

我在开源和Github profile方面的工作帮助我验证了我的经验，我得到的第一个机会是在 NED Karachi 的一个项目展览中担任评委，这是巴基斯坦工程和计算机科学领域的顶级大学之一。感谢 阿纳斯阿尤比 为我牵线萨姆拉赛义德和赛义德穆罕默德哈斯奈恩邀请我成为NED年度活动 ITEC 2018 的评委。

这是我第一次担任展览评委，我见证了许多有趣的项目，从人形说话机器人到生物和农业技术解决方案，到移动应用程序和游戏，再到数据科学实验。我特别喜欢的一个项目来自一个14岁的年轻程序员，他使用光子引擎创建了一个具有实时多人游戏功能的3D第一人称射击(FPS)游戏。我很惊讶一个14岁的孩子怎么能使用如此先进的工具来制作一个完美的游戏。

我想对穆罕默德·哈立德(Muhammad Khalid)大声欢呼，因为他给了我一整天美好的陪伴，在大学的道路和公园里漫步，在自助餐厅吃了一顿美好的午餐，随后喝了许多杯茶，还有无数的笑话和谈话。谢谢你，伙计，让那一天变得难忘。

此外，我还要特别感谢阿伊莎·坦维尔在活动中非常出色的欢迎招待会，帮助我与活动的组织者取得联系，并带我四处参观。我仍然对那天你的手机在人群中丢失感到非常难过，你因此非常紧张和分心。尽管如此，我还是很感激你那天抽出时间，尽管你那天担心死了。谢谢:)

在这次展览之后，我得到了一个关于一个我想都没想过的事件的演讲机会。那是由 Google开发者集团(GDG) Kolachi 举办的 Google IO Extended 2018 。再一次，非常感谢 Anas Ayubi 为我接通了 Eric Bhatti，他邀请我谈论“Android有什么新功能？”在Google IO Extended 2018。感谢埃里克的谈话。

作为一名android开发者，这对我来说意义重大。我非常紧张，因为这是我5年来第一次公开演讲，而且是在这么大的平台上。埃里克让我感到舒适，并相信我。令人惊讶的是，那天的情况变得非常奇怪。那天晚上我几乎没有睡上3个小时。我的演讲是在上午10:30。由于晚上没有睡觉，我不得不花4个小时直接去会场演讲。这是一个冒险的计划。我甚至不确定我是否能按时参加活动。埃里克又一次非常支持和配合。我参加了活动并做了演讲。主题是“Android有什么新功能？”。你可以在我的 SpeakerDeck个人资料 这里查看幻灯片。

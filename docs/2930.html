币安黑客事件对我们所有人来说都是一个令人担忧的发展

原文：https://medium.com/hackernoon/the-binance-hack-is-a-worrisome-development-for-us-all-75ec7a08df73#2019-05-08

最安全和领先的加密货币交易所之一，币安，刚刚被黑客入侵，报告超过 7000 个比特币被盗，价值 4050 万美元。

币安是一个高度可信的加密货币交易所，早些时候有过几次黑客攻击，但在这次之前都没有成功。据币安称，这次黑客不知何故获得了大量用户 API 密钥、2FA 代码和其他潜在信息。攻击的来源很多，币安认为包括网络钓鱼和病毒。

截至目前，只有一笔受影响的交易影响了币安的 BTC 热钱包，这只包含了他们全部 BTC 资产的 2%左右。据币安说，他们所有的其他钱包都是安全的，没有受到伤害。

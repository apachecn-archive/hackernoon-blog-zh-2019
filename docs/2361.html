用户生物概念—代码乒乓

原文：https://medium.com/hackernoon/user-bio-concept-code-pong-454be01ef91f#2019-04-12

戴恩·莱昂斯

大多数应用程序都有一些用户简介的变体。这些元素通常包含头像、名字、标题和其他相关元数据。所以我创建了一个代码笔项目，看看我们是否能提出一些新的概念。如果你想加入这个 code pong ，只需分叉项目并分享你的迭代。

声明:我们不打算在 Hacker Noon 2.0 上使用任何这些设计，但是如果任何人发布一些惊人的东西，我们一定会在使用之前征得同意。

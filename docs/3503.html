加密货币差异率

原文：https://medium.com/hackernoon/introducing-the-cryptocurrency-disparity-ratio-d2a41a4ee251?source=collection_archive---------7-----------------------#2019-06-06

基本面健康和市值严重错位的加密货币。

与我们在 CoinMarketCap 的朋友一起，每周我们都会制作加密差异比率。

加密差异率承认基本面健康( FCAS 评级)和当前市值之间的区别。高于1的分数反映资产被低估，低于1的分数表明资产被高估。

2019年6月6日:五大CDR 被低估加密资产:

  1. Remme (2.0)
  2. 早教网 (1.37)
  3. XYONetwork (1.35)
  4. 雷电网络 (1.33)
  5. 链家控股 (1.23)

2019年6月6日:五大CDR 被高估加密资产:

  1. 极光官员(AOA) (.29)
  2. Digix (.39)
  3. MonaCoin (.40)
  4. 华而不实
  5. 摩纳哥硬币(Crypto.com)(. 44)

根据FCAS评级，为了确定那些最被低估和高估的资产，当前市值被转换为标准化百分位数，以便可以在1000点的范围内直接进行比较。

[当前FCAS]与[当前标准化市值]的比率提供了一个相对的高/低分:

  • 值超过1表示资产被低估(即FCAS高于市值表示的值);
  • 低于1的值表示被高估的资产(其FCAS低于我们的预期)。

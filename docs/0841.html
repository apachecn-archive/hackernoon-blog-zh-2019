编程中的变革者:这个新的博客系列将聚焦于已经和/或继续对技术世界和编程世界产生积极影响的组织、协会和个人！

在第二篇文章中，我们将了解JavaScript的发明者。这是一个关于布伦丹内克的故事，他是一名程序员，拥有一长串不断增长的成就！

目前，艾希是 Brave 的首席执行官，该公司开发了一种新的互联网浏览器(名为Brave)，向用户承诺更多的隐私、更好的安全性和更快的浏览速度。但你可能也知道他是JavaScript的创造者和Mozilla的联合创始人。

在这篇博文中，我们将探索艾希是如何走到今天的，并讨论他的最新项目(Brave)以及它可能如何改变我们体验网络的方式。

艾希的背景

Brendan Eich在帕洛阿尔托长大，并在圣克拉拉大学获得了数学和计算机科学学士学位。之后，他于1985年在伊利诺伊大学香槟分校获得硕士学位。

他的编程生涯始于在一家名为 Silicon Graphics 的高性能计算制造商公司编写操作系统和网络代码。在Silicon Graphics工作了七年后，他转到了MicroUnity系统工程部门，编写微内核和DSP代码。MicroUnity是宽带微处理器技术的早期开发者，现在仍然存在。

网景和JavaScript时代

1995年4月，艾希受雇于网景通信公司，执行一项艰巨的任务:他必须为网景网络浏览器创建一种编程语言的工作原型；他只有10天时间来完成。

如果这听起来很疯狂，考虑一下风险:90年代中期的创新步伐非常快。微软刚刚发布了Windows 95，其中包括一个集成的Internet Explorer，对Netscape构成了严重威胁。

网景公司对他们的浏览器有一个稍微不同的看法。他们认为网络浏览器是一种新形式的分布式操作系统，而不是单一的应用程序。问题是HTML不足以满足他们的预期功能，所以他们需要一种新的语言，而且要快。

Java似乎是与Netscape集成的显而易见的选择，但是他们想要一种更轻量级的语言，让非专业人士也能使用。艾希在2012年的一次采访中讲述了当时的情况:

“如果我早在1995年5月就开始学习JavaScript，我会被告知它太像Java了，或者JavaScript正在与Java竞争……我接到营销指令，要让它看起来像Java，但又不至于太大……[它]需要成为一种愚蠢的小兄弟语言。”

Eich已经有了创造编程语言的历史。当他还是个学生的时候，他构建了新的语言来试验语法。在Silicon Graphics，他开发了用于构建网络监控工具扩展的新语言。事实证明，他是这项工作的合适人选，因为JavaScript是目前世界上使用最广泛的编程语言。

他在JavaScript方面的成功归结于这样一个事实，即它是一个简化的对象模型，并且结合了Java、C、SmallTalk和LISP的一些最佳特性。不仅如此，JavaScript被证明是适应性很强的。根据艾希的说法:

“[……]JavaScript一开始有足够多的好东西可以生存。如果你回想一下20世纪90年代，JavaScript被诅咒了，因为它主要用于浏览器底部状态栏中的小滚动消息或闪烁图像等烦人的事情。随着JavaScript在(20世纪90年代末)通过(ECMA)标准过程得到一些进化性的改进，它在2004年和2005年变得足够快和足够好，从而引发了Web 2.0革命。”(来源: IEEE )

共同创立Mozilla

1998年，Eich与Jamie Zawinski共同创立了Mozilla项目，管理Netscape源代码的开源贡献。

然而，事情并没有完全按照计划进行:AOL在1999年收购了网景，不久之后(2003年)关闭了网景浏览器。作为回应，艾希帮助将Mozilla基金会剥离出来，成为一个独立的实体。

2005年，艾希担任了Mozilla基金会的盈利机构Mozilla Corporation的首席技术官。他随后在2014年被提升为首席执行官，但他的任期很短。

勇敢建设:浏览网页的新方式

艾希目前是 Brave 的首席执行官，该公司开发了开源的Brave网络浏览器。

勇敢在几个方面是独一无二的。它旨在阻止广告和追踪器，以保护用户的数据并提高浏览速度(比Chrome快8倍)。然而，其最新颖的功能是微支付系统，允许用户通过观看内容获得奖励，并与内容创作者分享这些奖励。勇敢甚至有自己的货币:一种位于区块链的代币，叫做基本注意力代币。

BAT token于2017年5月31日通过破纪录的首次硬币发行 (ICO)推出。该公司通过在短短30秒内售出15亿个蝙蝠代币筹集了3500万美元——这是有记录以来最快的ICO。

Brave用户可以将他们的BAT钱包连接到他们的浏览器，并通过观看内部Brave网络上广告商推广的广告获得报酬。或者，用户可以选择使用Brave无广告，享受更无缝的浏览体验。

这种创新的收入分享模式可能会彻底改变我们体验网络的方式。如今，广告商向谷歌或脸书付费以接触消费者，谷歌和脸书利用消费者数据向合适的受众投放广告。但在Brave和BAT推出的模式下，用户保留了对其数据(和注意力)的控制权，并在选择与广告商分享数据时获得报酬。

想象一下，浏览时不必看到烦人的广告。想象一下，如果广告商真的为你看他们广告的时间付费(不仅仅是脸书或谷歌)。这两种选择现在都可以通过Brave实现。

# 什么是区块链周？

区块链周已经是第三年了，它是一系列的活动、聚会和会议，吸引了来自区块链和纽约的数千人。从初创公司和技术开发商到长期交易者和专家，区块链爱好者聚集在一起分享想法，讨论区块链的过去，现在和未来。

有许多锚定事件形成了区块链周的基础，如共识会议和以太。

在这一周，你可以看到数百场关于区块链、加密货币和任何你想要的加密数字的活动。我们按价格整理了活动列表，看看下面将要发生的事情。

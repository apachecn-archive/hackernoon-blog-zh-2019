介绍

随着区块链行业继续以极快的速度发展，跟上时代变得前所未有的重要。然而，对大多数投资者来说，知道在这方面该相信什么来源是困难所在。

数字证券市场将会大受欢迎，尽管目前几乎没有可操作的信息。出于这个原因，我们编辑了这项研究，包括事实、数据和见解，让你发掘它们的价值。最后，我们希望弄清楚为什么它们构成了一个可信的技术框架，以及它们与以前的行业标准有什么不同。

证券数字化

直到现在，传统金融还没有经历大规模的颠覆性变革，行业参与者需要找到创新的方法来扩大规模。

资产数字化(又名令牌化)正在创造一种全新的用户体验。例如，数字化资产和使用智能合约使股票所有者能够在投诉交易所按需出售私人证券股票，而不会遭受他们的资金被锁定在基金中多年的痛苦。

基本上，新技术提高了创建交易的便利性，即【流动性】。它没有对金融安全本身做任何事情，但它改变了所有权的管理方式以及投资后活动的自动化方式。

但是，大家也应该意识到，目前，数字化过程可能带来未知的风险。这就是为什么我们对数字证券市场中的一些领先公司进行了一系列采访，以补充我们的事实调查分析。我们采访了发行平台和流动性提供商、托管服务提供商、法律顾问、发行人和投资者，为您提供更多背景信息。你会发现他们的洞察力贯穿始终。

市场生态系统

像成功的公司一样，成功的生态系统会经历不同的阶段。数字证券市场在整个 2018 年都面临着这些主要问题:这些资产最终会成为更有声誉的生态系统中的新规范吗？它能吸引大量资金吗？为什么 s to 通常是更好的融资方式？根据开普勒金融公司的研究，去年通过一系列里程碑事件最终回答了这些问题。

数字证券生态系统是几个元素的组合，它们共同提供了从数字证券的创建到交易的整个活动范围。这些要素包括:代币发行、法律验证、法规遵从性、资产托管、代币交易流动性。所有这一切，包括发行，都将自动化。

协议正是让系统工作的原因。该协议保证代币根据用于筹资的任何豁免保持监管合规性。它管理谁可以相互交易，谁可以持有令牌。该协议还可以比传统生态系统更有效地拒绝尝试的交易，如果它将违反合规性的话。

市场上的领导者之一——secure ncy——正在开发一种可互操作的多账本协议，以保持链上和链下的合规性。它与遗留系统和 DLT 相互作用。在大多数情况下，公司专注于开发发行平台、代币协议或二级市场交易手段。另一方面，有些公司有一个完整的端到端平台，各方可以在全球范围内进行互动。

随着数字证券在金融市场的发展，一个清晰的基础设施开始形成。投资者正在采取行动，向支持数字证券发行、交易和托管的项目投入大量资金。 Harbor ，它提供了一个平台和合规协议，以规范私募证券在区块链的发行和交易方式，从主要的硅谷风险投资公司筹集了 2800 万美元的资金。证券化筹集了 1275 万美元，通过区块链技术将遗留证券数字化。根据开普勒金融数据库，已有超过 6.4 亿美元投资于开发旨在将资产和证券数字化主流化的技术的公司。全球合作伙伴网络正处于早期发展阶段。

我们将看到更多的经纪自营商、更多的交易所、更多的投资顾问、更多的律师和更多的众筹门户网站拥抱证券的令牌化。2019 年，市场将变得更加专业，规模更大，更具活力。

首先，有一个关于安全令牌的知识学习曲线需要你去克服。这不仅适用于团队本身，也适用于投资人。有些投资者在未知的环境中会感到不舒服，所以你需要注意并准备好教育他们。第二个主要缺点是不确定性。这适用于多种媒介。如果从法律的角度来看，所有的框架和法规都还在准备中。虽然这是一个新的资产类别，但它仍然按照旧的法律运作，这可能会产生一些冲突。我们看到这将是一个漫长的过程，让所有实体得出一个共同的结论，但这将是值得的。不确定性也存在于基础设施方面。尽管我们读到了安全令牌交易所的建立和法律的最终确定，但整个生态系统还远未准备好充分发挥作用。我们正处于早期阶段，每个人都还在为整个工作打下基础。所以很多事情都会出错。

法律视角

将证券数字化并使用智能合约可以自动执行适用的金融市场规则和法规。这些规则可以针对本地或全球市场进行编码，这是一个非常强大的概念。区块链有史以来第一次允许这种情况发生。但是下一阶段将会带来挑战。这需要时间，但这是事情的发展方向。列出整个 2018 年在法律里程碑方面取得的成就，我们希望有助于展示下一步的方向。

在一个不断发展的市场中，了解哪些国家的立法框架对 STO 最友好也很重要。我们邀请了 Security Token Lawyers 的联合创始人齐夫·凯南律师来澄清这个问题:

sto 与 ico 有很大不同，因为如果操作正确，在发行安全令牌时几乎没有监管风险。你可能会说 sto 是电子包装中的私募。结果是，你可以在任何允许出售私人证券的司法管辖区发行证券代币，包括美国和欧盟。有了 STOs，就没有必要跑到世界的边缘去寻找一个友好的司法管辖区。也就是说，除了遵守证券法之外，还有许多其他考虑因素。例如，一些银行不会与加密相关的基金合作，这意味着如果你的 STO 接受加密，那么你应该找到一个加密友好的银行。另一个考虑是税收。STO 代币的交易和流动性引发了一些问题，这些问题至今尚未完全解决。

词汇表

金融证券 —是可交易的金融资产，其价值来自对基础资产的合同要求，基础资产可以是实物或无形的。例如，商品和房地产是实物资产，但商品期货、交易所交易基金(ETF)和房地产投资信托基金(REITs)构成了金融资产，其价值取决于基础实物资产。金融资产通常比商品或房地产等其他有形资产更具流动性，可以在金融市场上交易。一般来说，证券是一种投资，是公司和其他商业企业筹集新资本的一种手段。

衍生品 —是一种金融工具，其价值来源于基础证券的价值。衍生品本身是两方或多方之间的合约，其价格由基础资产的波动决定。最常见的衍生工具包括期权、期货合同、远期合同、掉期、抵押债券(CMO)、权利和认股权证。

证券数字化(也称为证券代币化) —是以代表金融证券(基础)权利的区块链代币形式创建衍生工具的过程。数字化的目标是增加资本形成的流动性和效率，提高透明度和降低管理成本。

数字证券(也称为证券代币) —是区块链代币形式的衍生工具，代表金融证券(基础)的权利。使用分布式分类帐中记录的令牌进行的交易无法撤消或删除。对信息的访问可基于分类帐设置(公共、私有或混合)进行定制。

合规生态系统 —一整套程序、行动、活动、流程和各方，确保金融服务企业遵守法律法规和内部控制。将证券数字化并使用智能合约可以自动执行适用的金融市场规则和法规。

数字证券发行商 —一家直接向投资者提供数字证券为其运营融资的法律实体。发行人在法律上受发行规则、陈述和报告要求以及适用法规要求的与财务、重大发展和任何其他运营活动相关的其他契约的约束。

数字证券发行(“DSO”或证券代币发行，“STO”)—在完全合规的生态系统中，使用数字证券作为筹资工具的筹资类型。

数字证券发行平台 —一种技术平台，基于该平台的安全令牌标准，为发行人提供执行 DSO 的工具。

流动性 —资产或证券可以在市场上快速买卖而不影响证券价格的程度。有做市商在二级市场上提高证券的流动性。

做市商 —是银行、经纪公司或随时准备买卖证券的人。如果没有做市商，买家和卖家之间的匹配将需要长得多的时间，这将降低流动性，并可能增加交易成本，因为进入或退出头寸变得更加困难。
